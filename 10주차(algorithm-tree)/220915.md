# 기초수식 8번
- T(n) = T(n-1) + 1/n , T(1)=1 를 빅오표기법으로 표기하기

T(n-1) = T(n-2) + 1/(n-1)
T(n) = T(n-2)+ 1/(n-1) + 1/n
T(n-2) = T(n-3) + 1/(n-2)
T(n) = T(n-3) + 1/(n-2) + 1/(n-1) + 1/n
...
T(n) = 1/(n-k) + ...+ 1/(n-1) + 1/n

k = n-1

T(n) = 1 + ... + 1/(n-1) + 1/n
T(n) = Σ1~k 1/n ≈ ∫1~k 1/n ≈ logn
O(logn)
